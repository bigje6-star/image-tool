# ImageCraft - 专业图片处理工具

一个功能强大的在线图片处理工具，无需安装，直接在浏览器中使用。

## 功能特性

### 裁剪
- 预设比例：1:1、4:3、16:9、3:2
- 自由裁剪模式
- 拖拽调整裁剪框大小和位置

### 调整大小
- 宽度/高度精确调整
- 比例锁定开关
- 百分比快捷调整 (50%-200%)
- **实时文件大小预览**

### 去水印
- 涂抹笔工具
- 可调节笔刷大小
- 智能像素修复

### 文字添加
- 丰富的字体选择
  - 中文：思源黑体、思源宋体、站酷快乐体、站酷小薇、马善政毛笔，知行楷体等
  - 英文：Outfit、Poppins、Montserrat、Pacifico、Bebas Neue 等
- 7种文字效果：无、阴影、发光、描边、内阴影、渐变、3D
- 字号、颜色、对齐方式可选
- 面板确认/取消按钮，操作更便捷

### 导出
- 支持格式：PNG、JPEG、WebP、GIF
- JPEG/WebP 质量可调
- 保持原始尺寸或裁剪后尺寸

## 移动端支持

- 响应式布局，适配手机和平板
- 触摸操作支持
- 优化的触摸事件处理

## 使用方法

1. 打开 `index.html` 文件
2. 点击上传区域或拖拽图片上传
3. 选择工具进行编辑
4. 点击下载按钮导出图片

## 技术栈

- 纯前端实现 (Vanilla JavaScript)
- Canvas API 图片处理
- Google Fonts 字体
- 无需后端服务器
