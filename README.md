# ImageCraft - 专业图片处理工具

一个功能强大的在线图片处理工具，无需安装，直接在浏览器中使用。

## 功能特性

### 裁剪
- 预设比例：1:1、4:3、16:9、3:2
- 自由裁剪模式
- 拖拽调整裁剪框大小和位置

### 调整大小
- 宽度/高度精确调整
- 比例锁定开关
- 百分比快捷调整 (50%-200%)
- **实时文件大小预览**

### 去水印
- 涂抹笔工具
- 可调节笔刷大小
- 智能像素修复

### 文字添加
- 丰富的字体选择
  - 中文：思源黑体、思源宋体、站酷快乐体、站酷小薇、马善政毛笔，知行楷体等
  - 英文：Outfit、Poppins、Montserrat、Pacifico、Bebas Neue 等
- 7种文字效果：无、阴影、发光、描边、内阴影、渐变、3D
- 字号、颜色、对齐方式可选
- 面板确认/取消按钮，操作更便捷

### 导出
- 支持格式：PNG、JPEG、WebP、GIF
- JPEG/WebP 质量可调
- 保持原始尺寸或裁剪后尺寸

## 移动端支持

- 响应式布局，适配手机和平板
- 触摸操作支持
- 优化的触摸事件处理

## 使用方法

1. 打开 `index.html` 文件
2. 点击上传区域或拖拽图片上传
3. 选择工具进行编辑
4. 点击下载按钮导出图片

### 裁剪功能使用

1. 上传图片后，点击顶部工具栏的「裁剪」按钮
2. 在左侧面板选择裁剪比例（1:1、4:3、16:9、3:2 或自由）
3. 拖拽裁剪框调整位置和大小
4. 点击「应用裁剪」按钮确认

### 调整大小使用

1. 点击顶部工具栏的「调整大小」按钮
2. 在左侧面板输入目标宽度和高度
3. 开启比例锁定可保持宽高比
4. 使用快捷按钮或滑块快速调整缩放比例
5. 点击「应用调整」按钮确认

### 去水印使用

1. 点击顶部工具栏的「去水印」按钮
2. 在左侧面板调整笔刷大小
3. 在图片上涂抹需要去除的区域
4. 点击「撤销」按钮可回退操作

### 文字添加使用

1. 点击顶部工具栏的「添加文字」按钮
2. 在左侧面板设置文字样式：
   - 输入文字内容
   - 选择字体（支持中文和英文）
   - 调整字号大小
   - 选择文字颜色和描边颜色
   - 设置对齐方式（左/中/右）
   - 选择文字效果（无/阴影/发光/描边/内阴影/渐变/3D）
3. 点击「添加文字」按钮进入放置模式
4. 在图片上点击选择文字位置
5. 文字框会出现在点击位置，可直接输入文字
6. 点击右侧面板的「确认」保存文字，或「取消」放弃
7. 可以添加多个不同样式的文字
8. 点击「清除全部文字」删除所有已添加的文字

### 导出图片

1. 在右侧「导出设置」面板选择格式（PNG/JPEG/WebP/GIF）
2. 如果选择 JPEG 或 WebP，可调整图片质量
3. 点击「下载」按钮保存图片

## 技术栈

- 纯前端实现 (Vanilla JavaScript)
- Canvas API 图片处理
- Google Fonts 字体
- 无需后端服务器
